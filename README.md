# APP Dragon Ball Z

Aplicación móvil desarrollada en Android Studio, que consume una API REST para mostrar personajes de Dragon Ball Z

# Funcionalidades
- Consumo de API pública
- imágenes y datos
- Pantalla de bienvenida.
- Música de fondo en la pantalla principal
- Icono en la app
